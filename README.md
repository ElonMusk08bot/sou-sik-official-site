# Sou Sik Official Site

Next.js + Tailwind CSS official brand website for Sou Sik.

## Deploy
1. Upload this folder to GitHub.
2. Import the repository in Vercel.
3. Framework: Next.js.
4. Deploy.

## Edit
- Replace CTA URLs in `components/CTA.tsx` and pages.
- Replace image placeholders in `public/images/`.
- Add official product photos to `public/images/p1-hero.jpg`, `p1-detail.jpg`, etc.
